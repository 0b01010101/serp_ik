Синтаксис почти польностью похож на Small Basic.
Имеется три режима: интерактивный, обычный, чтение файла. Если строка начинается с номера, интерпретатор обрабатывает ее в обычном режиме. Если не с номера — в интерактивном. 
Если при запуске итерпретатора в качестве параметра был выбран файл, то он и будет исполняться.

    три команды: run (запускает программу), list (выводит исходник на экран), new (стирает программу);
    52 переменных (от A до z (asc2)): десятибайтовые целые числа без знака;
    выражения, которые могут включать в себя: числа, четыре арифметические операции, скобки, переменные, функции, массивы;
    три оператора: if, goto, =;
    две функции: print, input.
    поддержка локальных и глобальных массивов(длина имени до 127 символов), поддержка локальных переменных(длина имени до 127 символов) в функциях
    объявление функции происходит через оперетор "def", а массивов через "mas". Пример: def A() или  mas massive[7] 
    при объявлении фукции, в скобках можно указать количество параметров функции: def Z(3);
    перед наименованиями локальных переменных или массивов должен стоять символ "#": def S();
                                                                                       #locvar = 5;            -присвоение локальной переменной зачения пять
                                                                                       #mas massive[5];        -объявление локального массива размером в пять элементов(по 8 байт каждый)
                                                                                       #massive[3] = 6;        -присвоение третьему элементу локального массива значения шесть
                                                                                       #massive[2] = #locvar;  -присвоение значения #locvar
                                                                                    return #locvar;           -возврщает значение из #locvar
    название фунций это обязательно один символ от "A" до "Z"
    в конце определения функций должно быть "return;". После return может стоять как глобальная/локальная переменные, так и глобальный/локальный массив. В конце строки обязательно ";"!!!!!
    в качестве параметров фукции может быть любое выражение. В определении фукции параметры обозначаются как сивол "@", а за ним номер параметра: def D(2);
                                                                                                                                                   #a = @1;
                                                                                                                                                   #b = @2;
                                                                                                                                                 return @1 - @2;
                                                                                                                                                D(23, 4);
